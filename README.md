# Playlist Musical - TAD com Lista Sequencial

## Descrição

Este projeto implementa um TAD de Playlist Musical utilizando arrays em Java.

Os elementos armazenados representam IDs de músicas (tipo int).

## Estrutura

- PlaylistTAD.java → Interface
- PlaylistArray.java → Implementação com array
- Main.java → Classe de testes

## Operações implementadas

- Inserção no início
- Inserção no fim
- Inserção em posição
- Remoção no início
- Remoção no fim
- Remoção por posição
- Remoção por ID
- Busca por ID
- Verificação de estado
- Exibição da playlist

## Complexidade

Inserção no início: O(n)  
Inserção no fim: O(1)  
Remoção no início: O(n)  
Busca: O(n)
